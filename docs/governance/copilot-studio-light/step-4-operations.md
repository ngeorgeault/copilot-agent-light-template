# Étape 4 — Opérations & maintenance

Description :
- Supervision en production : télémétrie, monitoring, rapports d'utilisation.
- Processus de mise à jour : toute modification majeure du ZIP nécessite une nouvelle soumission et réévaluation.
- Revue périodique : ex. 6 mois (3 mois pour agents sensibles).

Procédures :
- Retrait d'un agent en cas d'incident
- Journalisation des versions et approbations dans SharePoint
- KPIs et reporting (nombre de soumissions, taux d'approbation, incidents)
