# Étape 6 — Revue par les administrateurs M365 / CoE

Description :
- Les administrateurs examinent la soumission et le contenu du ZIP (manifest, prompts, endpoints) en appliquant la checklist de sécurité et la grille de criticité.

Checklist (extrait) :
- Vérifier l'absence de PII exposée sans contrôle.
- Contrôler les connecteurs externes et les permissions demandées.
- Appliquer Sensitivity Labels, politiques DLP, et demander approbation sécurité si nécessaire.
- Décider du statut : Rejeter / Retour auteur / Approuver restreint / Approuver organisationnel.
