# Étape 0 — Création libre

Description :
- L'utilisateur métier utilise l'interface Microsoft 365 Copilot Chat (Copilot Studio Light) pour créer et tester un agent.
- L'interface permet l'export de l'agent au format ZIP (fichier d'agent) qui contient le manifest et les prompts.

Actions recommandées pour le créateur :
- Documenter l'objectif et les scénarios métier dans le ZIP ou dans le formulaire.
- Vérifier que les sources de connaissance n'exposent pas de données sensibles.
- Exporter le ZIP pour soumission via le formulaire officiel.
