# Étape 3 — Publication (via Applications Intégrées M365)

Description :
- Après approbation, les administrateurs publient l'agent via la gestion des Applications Intégrées de Microsoft 365.
- Appliquer paramètres : sensibilité, contrôle d'accès, DLP, consentements requis et portée de diffusion.

Actions admin :
- Créer l'Integrated App ou utiliser le mécanisme recommandé par Microsoft pour le déploiement
- Appliquer Sensitivity Label et politiques DLP
- Documenter le lien de publication dans la liste SharePoint
