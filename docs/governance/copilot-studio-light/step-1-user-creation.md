# Étape 1 — Utilisateur : Création dans Microsoft 365 Copilot

Description :
- L'utilisateur métier (non-développeur) utilise l'interface Microsoft 365 Copilot Chat pour créer un agent via Copilot Studio Light. Cette interface permet la conception déclarative d'un agent, la définition d'instructions, la configuration des sources de connaissance et des outils.

Références Microsoft Learn :
- https://learn.microsoft.com/fr-ca/microsoft-copilot-studio/guidance/

Actions recommandées pour le créateur :
- Définir clairement l'objectif métier et les scénarios ciblés.
- Rédiger des instructions claires, concises et testables (limiter la taille des instructions selon les recommandations de capacité).
- Sélectionner les sources de connaissance approuvées (SharePoint, OneDrive, etc.) et éviter d'inclure des sources non contrôlées.
- Ne pas inclure de données sensibles dans les prompts ou dans les sources non protégées.
- Exporter l'agent (ZIP) une fois la version prête pour soumission.

Métadonnées à documenter dans le formulaire de soumission :
- Titre, description, Product Owner, sources utilisées, indication PII, connecteurs externes requis, permissions demandées, public cible.
