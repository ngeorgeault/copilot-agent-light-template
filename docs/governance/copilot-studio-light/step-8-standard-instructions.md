# Étape 8 — Ajout du bloc d'instructions standard de l'entreprise

Description :
- Avant publication, ajouter un bloc d'instructions standard (template) imposé par l'entreprise aux instructions initiales : mentions de sécurité, limites d'usage, contacts pour incidents, et clauses de confidentialité.

Exemples de contenu du bloc standard :
- Ne pas divulguer de données personnelles non autorisées.
- En cas de doute, contacter le PO / sécurité.
- Les réponses générées sont à titre informatif et doivent être vérifiées pour les décisions critiques.
